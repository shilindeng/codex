# 图片系统

## 四维参数

- `theme`：内容主题领域
- `style`：视觉风格与表现方式
- `type`：图片用途
- `mood`：整体氛围与情绪强度

## 推荐组合

- 科技 + 未来科技 + 封面图 + 专业理性
- 商业 + 杂志封面 + 信息图 + 高级克制
- 职场 + 极简扁平 + 正文插图 + 高能激励
- 教育 + 手绘插画 + 信息图 + 温暖治愈
- 品牌 + 高级摄影 + 封面图 + 高级克制

## 默认图片规划

- 封面图：文章顶部视觉锚点，也作为微信封面候选
- 信息图：放在开头段落之后，用于解释核心框架
- 正文插图：按 H2/H3 小节插入，优先插在信息密度高的段落之后

## Prompt 组成

每张图的 Prompt 默认包含：

- 文章标题
- 核心观点
- 目标读者
- 图片用途
- 主题 / 风格 / 类型 / 氛围
- 构图要求
- 禁止事项

## `gemini-web` 说明

- 适合 Cookie / 浏览器会话场景
- 使用前必须完成同意检查
- 不应默认持久化原始 Cookie 字符串，除非用户明确要求
