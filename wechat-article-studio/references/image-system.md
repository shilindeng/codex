# 图片系统

## 四维控制参数

- `theme`：内容主题领域
- `style`：视觉风格
- `type`：图片用途
- `mood`：整体氛围

## 默认规划

- `1` 张封面图：仅用于封面和 `thumb_media_id`
- `1` 张信息图：优先放文末收束段
- `2~4` 张正文插图：按章节权重和信息密度插入
- 当正文字数 `> 2000` 时，总配图不少于 `5` 张；在默认策略下这意味着正文插图至少 `3` 张

## Prompt 组成

每张图默认包含：

- 文章标题
- 目标读者
- 图片用途
- 主题 / 风格 / 类型 / 氛围
- 章节焦点
- 禁止事项：过多小字、水印、无关 logo、无请求的人脸

## Provider 规则

- 自动默认只选官方图片后端：`gemini-api`、`openai-image`
- `gemini-web` 只在显式传入 `--provider gemini-web` 时启用
- 启用 `gemini-web` 之前必须先完成同意检查
